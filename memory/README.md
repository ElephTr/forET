# 会话与思考记录

本目录保存与 AI 助手的对话记录、思考过程和决策摘要。

## 目录结构

```
memory/
├── sessions/           # 会话摘要
│   └── YYYY-MM-DD-task-name.md
├── thoughts/           # 思考过程
│   └── topic-name.md
└── decisions/          # 关键决策记录
    └── decision-id.md
```

## 用途

1. **连续性**: 跨会话保持上下文
2. **复盘**: 回顾决策过程
3. **优化**: 基于历史改进
4. **备份**: 防止数据丢失

## 格式

所有记录使用 Markdown 格式，便于阅读和版本控制。
