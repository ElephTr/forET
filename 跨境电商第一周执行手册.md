## Day 2：建站准备

### 上午任务（3小时）

#### 2.1 域名购买

**购买流程：**
```
□ 访问 namecheap.com 或 godaddy.com
□ 搜索品牌名+.com
□ 检查是否可用
□ 添加到购物车
□ 使用优惠码（搜索"Namecheap coupon"）
□ 完成购买（建议买Privacy Protection）
□ 记录域名管理后台登录信息
```

**域名选择原则：**
```
✅ 简短易记（<15个字符）
✅ 容易拼写
✅ .com优先
✅ 与品牌相关
✅ 避免数字和连字符

❌ 不要侵权（检查商标）
❌ 不要太长
❌ 不要难拼写
```

#### 2.2 Shopify注册与基础设置

**注册流程：**
```
□ 访问 shopify.com
□ 点击"Start free trial"
□ 填写邮箱、密码、店铺名
□ 回答基础问题（可跳过）
□ 进入后台
```

**基础设置：**
```
□ Settings > General
  - 店铺名称
  - 邮箱地址
  - 时区（目标市场时区）
  - 货币（USD）

□ Settings > Payments
  - 激活PayPal
  - 激活Shopify Payments（如有）
  - 设置测试模式（测试用）

□ Settings > Shipping
  - 删除默认运费模板
  - 创建新模板：
    * 国家：目标市场（如US/UK）
    * 运费：$0（包邮）或实际运费

□ Settings > Taxes
  - 设置自动计算税费

□ Settings > Notifications
  - 自定义邮件模板（可选）
```

## Day 3：产品上架

### 上午任务（3小时）

#### 3.1 产品信息准备

**产品文案撰写（用AI辅助）：**

**标题公式：**
```
[品牌] + [核心关键词] + [关键特性] + [适用场景]

示例：
PetPlus Smart Automatic Pet Feeder with App Control for Cats and Dogs
```

**产品描述结构：**
```
1. Hook（吸引注意）- 1句痛点
2. 产品核心卖点 - 3-5个bullet points
3. 详细描述 - 解决什么问题
4. 产品规格 - 尺寸/重量/材质
5. 使用场景 - 图片/视频
6. 社会证明 - 评价/销量
7. 行动号召 - 购买按钮
```

**AI提示词（用ChatGPT/Kimi）：**
```
请帮我写一篇跨境电商产品描述。

产品：智能宠物喂食器
目标市场：美国
目标客户：25-40岁，有猫或狗的上班族

要求：
1. 标题包含关键词，吸引人
2. 开头用痛点引入
3. 列出5个核心卖点（用bullet points）
4. 语言简洁有力，适合美国消费者
5. 包含行动号召
6. 字数500-800词
```

#### 3.2 产品图片准备

**图片来源：**
```
方案A：向供应商要
□ 联系供应商索要高清图片
□ 要求：白底图+场景图+细节图

方案B：自己拍摄
□ 购买样品（Day 1联系供应商）
□ 用手机拍摄（自然光）
□ 背景：白色或场景化

方案C：AI生成/修改
□ 用Canva设计
□ 用Remove.bg去背景
□ 用Photopea编辑（免费）
```

**图片要求：**
```
□ 主图：白底，产品占80%，1000x1000px
□ 场景图：产品使用场景， lifestyle
□ 细节图：产品特写，展示质感
□ 尺寸图：标注尺寸
□ 包装图：展示包装
□ 视频：15-30秒使用视频
```

### 下午任务（3小时）

#### 3.3 上架产品

**Shopify上架流程：**
```
□ Products > Add product
□ 填写产品标题
□ 填写产品描述
□ 上传产品图片
□ 设置价格（成本x2.5-3倍）
□ 设置库存（9999，虚拟库存）
□ 设置重量（用于计算运费）
□ 设置产品类型和标签
□ 设置SEO标题和描述
□ 保存
```

**定价策略：**
```
售价 = 产品成本 + 物流成本 + 广告成本 + 利润

示例：
- 产品成本：￥85
- 物流成本：￥35（一件代发）
- 目标毛利率：50%
- 售价计算：(85+35)/0.5 = ￥240 ≈ $35

检查竞品价格：$30-50
定价：$39.99（心理定价）
```

**首批上架：**
```
□ 上架3-5个产品
□ 每个产品完整信息
□ 测试购买流程
□ 检查移动端显示
```

## 常见问题解决

### Q1：供应商不回复？
**解决：** 
- 多发几家，不要吊死在一棵树上
- 用中文沟通，表明诚意
- 直接电话沟通（1688有电话）

### Q2：网站加载慢？
**解决：**
- 压缩所有图片（<200KB）
- 减少插件数量
- 使用CDN加速

### Q3：不知道拍什么内容？
