## 一、小红书文案生成

### 模板1：干货教程类

```
你是一位[领域]专家博主，请帮我写一篇小红书教程笔记。

主题：[具体主题]

要求：
1. 标题格式：数字+痛点+解决方案（如：3个技巧让你XXX）
2. 开头：用痛点或好奇引入（50字内）
3. 正文：分3-5点，每点包含：
   - 小标题（加emoji）
   - 具体方法（简洁明了）
   - 效果/案例（可选）
4. 结尾：总结+引导互动（提问或求收藏）
5. 添加相关话题标签（5个）

语气：亲切、实用、像朋友分享
字数：400-600字
```

**示例：**
```
主题：如何用AI写小红书爆款文案

你是一位AI工具博主，请帮我写一篇小红书教程笔记。

主题：如何用AI写小红书爆款文案

要求：
1. 标题格式：数字+痛点+解决方案
2. 开头：用痛点或好奇引入（50字内）
3. 正文：分4点，每点包含小标题（加emoji）、具体方法
4. 结尾：总结+引导互动
5. 添加相关话题标签（5个）

语气：亲切、实用、像朋友分享
字数：400-600字
```

## 二、标题生成

### 模板：批量生成标题

```
你是一位小红书标题专家，请帮我生成10个吸引人的标题。

内容主题：[主题]
内容类型：[干货/故事/推荐/观点]
目标用户：[描述]

要求：
1. 使用数字（3个、5步、7天等）
2. 包含情绪词（绝了、救命、真的、后悔等）
3. 制造好奇或悬念
4. 突出痛点或利益
5. 控制在20字以内

格式：直接列出10个标题
```

**标题公式参考：**

| 公式 | 示例 |
|------|------|
| 数字+痛点+解决方案 | 3个技巧让你告别拖延症 |
| 身份+结果+方法 | 打工人如何用AI月入过万 |
| 情绪+现象+观点 | 真的绝了！原来这才是正确做法 |
| 对比+转折 | 从月薪3千到3万，我只做了这一件事 |
| 提问+悬念 | 为什么你努力了还是赚不到钱？ |
| 禁忌+揭秘 | 千万别这样做！90%的人都错了 |

## 十、使用技巧

### 10.1 提示词优化原则

1. **具体 > 笼统**
   - ❌ "写一篇好文章"
   - ✅ "写一篇关于AI写作技巧的小红书干货文，分5点说明，每点配emoji"

2. **结构化 > 自由发挥**
   - 给AI明确的结构框架
   - 指定字数、格式、风格

3. **迭代优化**
   - 第一次生成后，根据结果调整提示词
   - 多次迭代，直到满意

### 10.2 常见错误

| 错误 | 正确做法 |
|------|----------|
| 提示词太短 | 提供足够上下文 |
| 要求太模糊 | 明确具体标准 |
| 一次要太多 | 分步骤、分批次 |
| 不检查直接发 | 人工审核+修改 |
| 完全依赖AI | AI生成+人工优化 |

### 10.3 效率提升技巧

```
1. 建立提示词库（收藏本文档）
2. 批量生成，批量处理
3. 保存好用的提示词，重复使用
4. 根据平台调性调整语气
5. 记录效果好的提示词，形成个人模板
```
