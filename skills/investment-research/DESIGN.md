# 投资研究 Skill - 第一性原理设计文档

## 核心问题：投资本质上是什么？

投资 = 用今天的资金换取未来更高的购买力
关键 = 预测未来 + 控制风险

## 信息获取的第一性原理

### 1. 价格信息（市场正在发生什么）
- **实时行情**：股价、成交量、资金流向
- **市场结构**：板块轮动、市场情绪、波动率
- **衍生品市场**：期权隐含波动率、期货升贴水

### 2. 基本面信息（价值在哪里）
- **宏观**：利率、通胀、GDP、就业、政策
- **行业**：景气度、竞争格局、技术变革
- **公司**：财报、管理层、护城河、估值

### 3. 情绪与行为（别人怎么想）
- **新闻舆情**：主流媒体、社交媒体、搜索趋势
- **机构动向**：持仓变化、研报评级、调研记录
- **散户情绪**：开户数、融资余额、恐慌/贪婪指数

### 4. 跨市场信号（全球联动）
- **汇率**：美元指数、人民币汇率
- **大宗商品**：原油、黄金、铜（经济晴雨表）
- **全球市场**：美股、欧股、新兴市场、加密货币

## 信息处理的第一性原理

### 什么信息真正重要？

| 时间维度 | 关键信息 | 决策价值 |
|---------|---------|---------|
| 短期(天) | 资金流向、突发事件、技术形态 | 交易时机 |
| 中期(月) | 业绩预期、政策变化、行业趋势 | 仓位调整 |
| 长期(年) | 经济结构、技术革命、人口变化 | 资产配置 |

### 信息质量评估
- **及时性**：新闻 vs 财报 vs 调研
- **权威性**：官方数据 > 机构研报 > 媒体解读 > 社交媒体
- **独立性**：多方验证，避免信息茧房
- **可操作性**：能转化为具体决策的信息才有价值

## 分析框架

### 自上而下 (Top-Down)
1. 全球经济周期判断
2. 资产类别配置（股/债/商/现）
3. 行业/地区选择
4. 具体标的选择

### 自下而上 (Bottom-Up)
1. 筛选被低估/高估标的
2. 深度基本面分析
3. 构建组合对冲风险

### 量化信号
- 估值分位数（PE/PB历史位置）
- 技术指标（趋势、动量、均值回归）
- 资金流向（北向资金、主力资金）
- 情绪指标（VIX、融资余额）

## 输出格式

### 日常简报 (Daily Brief)
- 市场概览：主要指数涨跌、成交量
- 重要新闻：政策、宏观数据、公司公告
- 资金流向：板块轮动、北向资金
- 情绪指标：恐慌/贪婪指数、VIX
- 今日关注：重要事件、数据发布

### 周报 (Weekly Report)
- 本周回顾：市场走势、关键事件
- 数据更新：重要经济指标
- 行业观察：景气度变化、政策影响
- 下周展望：关注事项、风险提示

### 深度分析 (Deep Dive)
- 主题研究：特定行业/公司/策略
- 估值分析：历史分位、同业对比
- 风险评估：下行空间、黑天鹅
- 投资建议：买入/持有/卖出 + 理由

## 数据源规划

### 免费数据源
- 行情：Yahoo Finance, Alpha Vantage, 东方财富
- 宏观：FRED, 国家统计局, Wind（部分免费）
- 新闻：RSS订阅、官方媒体API
- 情绪：Google Trends, 微博热搜

### 付费数据源（可选）
- Wind/同花顺iFinD：专业金融数据
- Bloomberg/Reuters：机构级新闻
- 另类数据：卫星、信用卡、招聘数据

## 技术实现

### 数据采集
- API 调用（REST/WebSocket）
- RSS 订阅
- 网页抓取（谨慎使用）

### 数据存储
- 时序数据库（行情数据）
- 文档存储（新闻、研报）

- 可视化：matplotlib, plotly

## 风险控制第一性原理

1. **永远不要All-in**：分散是唯一的免费午餐
2. **止损比止盈更重要**：控制下行，让上行自己跑
3. **不要预测，要应对**：准备多种情景，而不是押注单一结果
4. **了解你的对手**：市场由人组成，理解行为金融
5. **保持谦逊**：承认自己不知道的事情

## 这个 Skill 的核心价值

不是给你"买什么"的答案，而是：
1. **系统性信息收集**：帮你监控应该关注的信息
2. **结构化分析框架**：提供思考投资的逻辑
3. **决策辅助**：给出基于数据的观点，供你参考
4. **持续学习**：记录预测vs结果，迭代改进

最终决策权永远在你。
