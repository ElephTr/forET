# 商业机会分析 Skill

基于第一性原理的商业机会发现与分析系统。

## 核心哲学

做生意 = 解决他人问题 × 他人付费意愿 × 可持续性

我们不预测风口，而是：
1. **系统扫描**市场真实需求
2. **结构化分析**商业逻辑
3. **数据验证**机会可行性
4. **持续迭代**商业模式

## 使用方法

### 发现商业机会
```
@business-opportunity scan [领域/行业]
```
扫描特定领域的商业机会。

### 分析商业模式
```
@business-opportunity analyze [商业想法]
```
分析具体商业模式的可行性。

### 竞争分析
```
@business-opportunity competition [行业/产品]
```
分析竞争格局和市场空间。

### 趋势洞察
```
@business-opportunity trend [主题]
```
分析行业趋势和机会窗口。

## 分析框架

### 1. 需求验证 (Demand Validation)
- 问题是否真实存在？
- 目标用户是谁？
- 他们现在怎么解决？
- 付费意愿有多强？

### 2. 市场 sizing (Market Sizing)
- TAM (Total Addressable Market)
- SAM (Serviceable Addressable Market)
- SOM (Serviceable Obtainable Market)

### 3. 竞争分析 (Competition)
- 现有解决方案
- 竞争优势/劣势
- 差异化机会

### 4. 商业模式 (Business Model)
- 收入来源
- 成本结构
- 盈利路径

### 5. 可行性评估 (Feasibility)
- 启动成本
- 技能要求
- 时间投入
- 风险评估

## 数据源

- **搜索趋势**: Google Trends, 百度指数
- **社交媒体**: Reddit, Twitter, 知乎, 小红书
- **电商平台**: Amazon, 淘宝, 京东 (销量/评价)
- **招聘数据**: 职位需求反映行业热度
- **投资数据**: 融资趋势反映资本关注

## 输出原则

1. **从需求出发**，而非从产品出发
2. **数据支撑**，而非主观臆断
3. **具体可执行**，而非泛泛而谈
4. **风险明示**，而非只谈机会

## 免责声明

本 skill 提供的分析仅供参考，不构成投资建议。
商业决策需结合自身情况，过往数据不代表未来表现。
