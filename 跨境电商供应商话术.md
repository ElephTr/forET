# 跨境电商 - 供应商谈判话术与模板

> 与1688供应商沟通的标准话术和流程

---

## 一、首次联系话术

### 模板1：标准询价

```
你好，我对你们的产品很感兴趣，想了解一下：

1. 产品型号：[具体型号或链接]
2. 单价是多少？
3. 起订量（MOQ）是多少？
4. 是否支持一件代发？
5. 能否提供产品图片和视频？
6. 发货到海外（美国/欧洲）的物流方式和费用？
7. 生产周期是多久？

我们是做跨境电商的，如果合作顺利，后续会有稳定订单。

期待你的回复，谢谢！
```

### 模板2：英文询价（如需）

```
Hi,

I'm interested in your product [product name/link]. 
Could you please provide the following information:

1. Unit price for [quantity] pieces?
2. Minimum Order Quantity (MOQ)?
3. Do you support dropshipping?
4. Can you provide product images and videos?
5. Shipping options and costs to [country]?
6. Production lead time?

We are doing cross-border e-commerce business. 
If everything goes well, we will have regular orders.

Looking forward to your reply.

Best regards,
[Your name]
```

### 模板3：快速筛选

```
你好，我想快速了解：

产品：[链接]
问题：
1. 一件代发价格？
2. 能否提供英文产品资料？
3. 能否直接发海外？

如果以上都可以，我们再详细聊。
```

## 二、深度沟通话术

### 确认合作意向后

```
你好，感谢回复。我想进一步确认合作细节：

【产品信息】
- 产品型号：____
- 一件代发价格：____
- 产品重量/尺寸：____

【包装】
- 产品包装是否中性（无中文/供应商信息）？
- 能否定制包装（加我的Logo）？MOQ多少？
- 包装尺寸和重量：____

【物流】
- 支持哪些物流方式？（云途/燕文/4PX等）
- 到美国的运费大约多少？时效多久？
- 是否提供物流追踪号？

【资料】
- 能否提供高清产品图片（白底图+场景图）？
- 能否提供产品视频？
- 能否提供英文产品说明书？

【售后】
- 产品质量问题如何处理？
- 退换货政策是怎样的？
- 产品保修期多久？

【付款】
- 支持哪些付款方式？
- 账期如何？

期待详细回复，谢谢！
```

## 三、样品申请话术

### 模板1：申请样品

```
你好，我想先购买样品测试质量。

需要样品：
- 产品：[型号]
- 数量：1-2个
- 地址：[你的地址]

请问：
1. 样品价格是多少？（是否按零售价？）
2. 样品费后续大货能否抵扣？
3. 样品发货时效多久？
4. 能否顺便发几个不同款式的样品？

如果样品质量OK，我们会尽快下单。
```

### 模板2：样品反馈

```
你好，样品已收到，测试反馈如下：

【优点】
- ____
- ____

【问题】
- ____
- ____

【建议】
- ____

请问这些问题能否改进？改进后的价格是多少？
如果没问题，我们准备下第一批订单。
```

## 四、订单沟通话术

### 正式下单

```
你好，我准备下第一批订单：

【订单信息】
产品：[型号]
数量：[数量]
价格：[单价]
总金额：____

【物流信息】
发货方式：一件代发/批量发货
收货地址：____
收件人：____
电话：____

【包装要求】
□ 中性包装（无中文）
□ 贴[SKU标签/我的Logo]
□ 其他要求：____

【资料】
请随货提供：
- 产品图片（高清）
- 产品视频
- 英文说明书（如有）

【付款】
请提供付款方式，我安排付款。

预计什么时候可以发货？
```

### 跟进发货

```
你好，订单[订单号]已付款，请确认：

1. 款项是否收到？
2. 预计什么时候发货？
3. 发货后请提供物流单号
4. 能否拍一下发货前的产品照片？

谢谢！
```

## 五、售后问题处理话术

### 质量问题

```
你好，客户反馈产品有以下问题：

【问题描述】
- 问题1：____
- 问题2：____

【证据】
- 客户提供的照片/视频：[附件]

【客户诉求】
- 退款/换货/补偿

请问这种情况如何处理？
是我们的责任还是物流的责任？
后续如何避免类似问题？

期待解决方案，谢谢！
```

### 物流延误

```
你好，订单[单号]的物流显示延误：

- 发货时间：____
- 当前状态：____
- 预计延误：____天

客户很着急，能否：
1. 查询具体原因？
2. 催促物流加快？
3. 如果延误太久，能否补发或退款？

请尽快处理，谢谢！
```

## 六、长期合作话术

### 谈判更好价格

```
你好，合作一段时间了，想谈一下长期合作的价格：

【目前情况】
- 合作时间：____个月
- 累计订单：____单
- 累计金额：____元

【期望】
基于以上订单量，能否给予更优惠的价格？

【承诺】
如果价格合适，我们承诺：
- 每月稳定订单____单
- 优先选择你们家
- 长期合作关系

请问能做到什么价格？
```

### 定制/OEM合作

```
你好，我们想做自己的品牌，想咨询OEM合作：

【需求】
- 产品：基于[现有型号]定制
- 定制内容：
  * 外观：[颜色/材质/设计]
  * 包装：定制包装盒+说明书
  * Logo：印刷Logo
  * 功能：[如有改动]

【数量】
- 首批：____个
- 后续：每月____个

【问题】
1. 能否做OEM？
2. MOQ多少？
3. 打样费用和时间？
4. 大货价格和交期？
5. 能否签保密协议？

请提供详细方案，谢谢！
```

## 八、注意事项

### ✅ 应该做的

1. **多找几家供应商对比**
   - 至少联系10-15家
   - 选择2-3家备用

2. **小批量测试**
   - 首次不要下大单
   - 验证质量和服务

3. **保留证据**
   - 聊天记录保存
   - 合同/协议书面确认
   - 付款凭证保留

4. **建立关系**
   - 节日问候
   - 定期沟通
   - 长期合作心态

### ❌ 不应该做的

1. **不要只图便宜**
   - 价格过低可能质量差
   - 综合考虑性价比

2. **不要一次性付款**
   - 大额订单分阶段付款
   - 30%定金+70%发货前

3. **不要忽视合同**
   - 明确质量标准和售后
   - 违约责任要写清楚

4. **不要完全依赖一家**
   - 至少2-3家供应商
   - 防止断供风险

## 九、常用术语
